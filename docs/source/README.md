# Source corpus

Raw source corpus files are intentionally not committed to the public GitHub repository.

Excluded from Git:

- course slide decks under `docs/source/course-materials/`,
- extracted course text under `docs/source/extracted-text/`,
- private project brief files under `docs/source/project-brief/`.

The committed `docs/` files contain derived project documentation, product boundaries, implementation plans, QA reports, and safety notes. If a future private repository or Git LFS setup is approved, raw corpus files can be shared there with the right permissions.
